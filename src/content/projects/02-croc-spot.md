Croc-Spot is an innovative web-based platform designed to enhance crocodile safety in the Northern Territory. It provides real-time monitoring through multiple live streams, AI-powered recognition, and crowd-sourced data.

## Features

Users interact with live footage, report sightings, and receive timely alerts about crocodile activity in high-risk areas. The platform combines cutting-edge AI technology with community engagement to improve public safety.

## Technology Stack

Built with Vue.js for the frontend interface, D3.js for interactive data visualizations, and Python with TensorFlow for AI-powered crocodile detection. The entire platform is hosted on AWS for reliability and scalability.

## Impact

This project was recognized as a GovHack winner, demonstrating innovation in using technology to solve real-world safety challenges in the Northern Territory.
