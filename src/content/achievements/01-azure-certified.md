Professional certification validating expertise in cloud concepts, core Azure services, plus Azure management and governance features and tools.

This certification demonstrates a solid understanding of cloud computing concepts and Microsoft Azure services, including:

- Core Azure services and solutions
- Azure pricing, SLA, and lifecycle management
- Cloud security, privacy, compliance, and trust
- Azure governance and management tools

Issued by Microsoft, this certification validates foundational knowledge of cloud services and how those services are provided with Microsoft Azure.
